団体について

NPO法人くりあーは、地域で暮らし続けたいという当事者・家族の願いを起点に設立されました。 - 社会課題を「個人の困りごと」に矮小化せず、地域の仕組みとして解決することを目指しています。 + 社会課題を個人だけの問題にせず、地域の仕組みとして支えることを重視しています。

理念・設立趣旨

- だれもが尊厳を持って暮らせる地域社会を実現するため、必要な支援を切れ目なく提供し、 - 支援する側・される側を固定しない関係づくりを進めます。 + 必要な支援を切れ目なく届けるだけでなく、支援する側・される側を固定しない関係づくりを進め、 + 相互に支え合えるコミュニティを育てます。

療育支援事業

+
+
+

事業の目的

+

発達特性に応じた支援を通じ、子どもと家族が地域で安心して生活できる基盤を整えます。

+
+
+

対象者

+

自閉症児・者およびその家族、支援機関と連携が必要な地域関係者。

+
+
+

具体的な内容

+

個別支援、小集団プログラム、保護者面談、家庭・学校・関係機関との連携支援。

+
+
+

実施頻度・場所

+

平日・土曜を中心に実施（詳細は活動報告で案内）。岡山県赤磐市を中心に展開予定。

+
+
+

今後の展開: 支援記録の可視化と支援ネットワーク拡充により、継続的な伴走支援体制を強化します。

+
-

ボランティア参加

イベント運営や地域食堂など、現場での実践に参加できます。

相談支援事業

+
+
+

事業の目的

+

本人・家族の不安や課題を整理し、適切な支援資源につなげることで生活の安定を図ります。

+
+
+

対象者

+

自閉症児・者とその家族、福祉・教育制度の活用に悩む地域住民。

+
+
+

具体的な内容

+

初回相談、継続相談、制度利用の整理、関係機関への同行・連絡調整。

+
+
+

実施頻度・場所

+

予約制で随時対応。対面・オンラインの併用を予定しています。

+
+
+

今後の展開: 地域内の相談窓口連携を強化し、切れ目のない支援導線を整備します。
